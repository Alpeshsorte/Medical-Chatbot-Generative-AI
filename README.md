# 🏥 Medical Chatbot

A Medical Chatbot is an AI-powered healthcare assistant that helps users get basic medical information, understand symptoms, learn about diseases, and receive general health guidance through a conversational interface. This project is built using Python and provides quick, user-friendly responses while encouraging users to consult healthcare professionals for medical emergencies.

---

## 📌 Features

- 💬 Interactive chatbot interface
- 🤒 Symptom-based health guidance
- 💊 Medicine and disease information
- 🔍 Search health-related queries
- 📖 Frequently Asked Questions (FAQ)
- 👤 User-friendly interface
- 🔒 Secure and lightweight application
- ⚡ Fast response generation

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/Medical-Chatbot.git
```

### 2. Navigate to the Project Folder

```bash
cd Medical-Chatbot
```

### 3. Create a Virtual Environment (Optional)

```bash
python -m venv venv
```

Activate the environment:

**Windows**

```bash
venv\Scripts\activate
```

**Linux/macOS**

```bash
source venv/bin/activate
```

### 4. Install Dependencies

```bash
pip install -r requirements.txt
```

### 5. Run the Application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
